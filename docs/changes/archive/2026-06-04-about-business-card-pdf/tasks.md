- [x] 4.1 Add collapsed-state detection to `BusinessCard`: in `useEffect`, compute `triggerY = wrapper.getBoundingClientRect().top + window.scrollY - navBottom` once at mount. Attach a passive `scroll` event listener on `window`; when `scrollY >= triggerY`, set `data-collapsed="true"` on the wrapper; otherwise remove it. Verify: `data-collapsed="true"` appears when the card top reaches the navbar bottom; removed on scroll back to top. Satisfies: collapsed header design decision. **[Implementation note: originally used IntersectionObserver with a sentinel div.
